# Bin files loader

דוגמה פשוטה ליישום Flet הטוען קובץ טיסה בפורמט CSV, מפענח אותו לנקודות קואורדינטות
ומציג את המסלול על מפה של OpenStreetMap. הפרויקט מיועד ללמידה ולכן שומר על מבנה
קוד מינימלי וברור.

## הפעלה

```bash
pip install -r requirements.txt
flet run gui/app.py
```

במסך הראשי לחץ על "בחר קובץ" ובחר קובץ טקסט שבו כל שורה מכילה את הערכים הבאים:

```
lat,lon,altitude_m,timestamp_s[,roll,pitch,yaw]
```

לאחר טעינת הקובץ האפליקציה תציג את המסלול על המפה.
