# Bin files loader

דוגמה פשוטה ליישום Flet הטוען קובץ טיסה של ArduPilot בפורמט BIN, מפענח אותו
לנקודות קואורדינטות ומציג את המסלול על מפה של OpenStreetMap. הפרויקט מיועד
ללמידה ולכן שומר על מבנה קוד מינימלי וברור.

## הפעלה

```bash
pip install -r requirements.txt
flet run gui/main.py
```

במסך הראשי לחץ על "בחר קובץ" ובחר קובץ ``.BIN`` שיצא מבקר הטיסה. האפליקציה
מפענחת את הודעות ה-GPS שבתוך הקובץ ומציגה את המסלול על המפה. למטרות פיתוח
ניתן גם לטעון קובץ טקסט פשוט שבו כל שורה היא: ``lat,lon,altitude_m,timestamp_s``.
